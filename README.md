# 🥬 VeggieGo - Super Fast Fruit & Vegetable Delivery

VeggieGo is a modern web application that provides lightning-fast delivery of fresh fruits and vegetables right to your doorstep. Built with cutting-edge technologies, VeggieGo ensures a seamless shopping experience with real-time order tracking and instant delivery updates.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) - React framework for production
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) - Lightweight state management
- **AI Integration**: [Google Gemini](https://deepmind.google/technologies/gemini/) - For intelligent product recommendations and customer support
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) - Beautiful and accessible UI components
- **Styling**: Tailwind CSS for modern, responsive design

## 🛠️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/veggie.git
cd veggie
```

2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add your environment variables:

```env
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key
```

4. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
